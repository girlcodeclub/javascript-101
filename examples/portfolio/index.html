About JavaScript

What is JavaScript?

JavaScript (or "JS") is a programming language that makes websites interactive. If HTML is the "skeleton" of a webpage and CSS is the "skin", then JavaScript is the "muscles" that make it move. It's one of the most popular languages in the world.

Why We're Learning It

  • It's the only language that runs in all web browsers.
  • You can use it to build websites, apps, and even games.
  • It's a great "first" text-based language after Scratch.
  • You can use it to build server-side code (with Node.js).

How It Relates to Scratch

  1. Scratch uses visual blocks; JavaScript uses text commands.
  2. "When Green Flag Clicked" is an **Event**, just like `onclick` in JS.
  3. "Make a Variable" is a **Variable**, just like `let myName = "Thalia"` in JS.
  4. "If/Then" blocks are **Conditionals**, just like `if ( ... ) { ... }` in JS.

Where to Learn More

The best professional resource for JavaScript is the Mozilla Developer Network (MDN).
